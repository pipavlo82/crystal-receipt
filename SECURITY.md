# SECURITY

## Security status

This project is **not** production security-certified.

Crystal Receipt contains proof-substrate, verification, documentation, and static viewer work that may still change rapidly. Treat the repository as experimental software unless explicitly stated otherwise.

## Reporting security issues

Please report security issues **privately** through GitHub Security Advisories if that feature is enabled for this repository.

If GitHub Security Advisories are not available, open a **minimal non-exploit public issue** instead.

When reporting publicly:
- do **not** include secrets
- do **not** include exploit details
- do **not** include step-by-step attack instructions
- keep the report limited to enough information for maintainers to reproduce and triage safely

## Supported version

For now, the only supported version is:
- `main`

Security fixes, review, and triage should be assumed to target `main` first.
